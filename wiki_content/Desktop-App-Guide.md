## Core Features
- **One-Click Sync**: "Activate" cloud projects to download them, or "Deactivate" local projects to offload them to the cloud.
- **Software Management**: Link specific installed apps to projects. The app can auto-uninstall software when it's no longer needed by any active project.
- **Portable Mode**: Run the entire suite from a USB drive or cloud folder.
