# Remote Agent

The **Remote Agent** is a FastAPI-based Python server that runs on your Windows workstation. It acts as the secure gateway between your mobile devices and your development environment.

## Primary Functions
- **Project Management**: High-speed movement of project files between Local and Cloud storage.
- **Shell Bridge**: Real-time execution of PowerShell/CMD commands via WebSockets.
- **Automation**: Automatic software installation via WinGet and external asset handling.

For technical details on how the agent manages files, see the [[Remote Agent Deep Dive]].
