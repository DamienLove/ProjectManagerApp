# Android Client

The **Android Client** is a native application built with Jetpack Compose that gives you full control over your workstation from your phone.

## Key Features
- **Project Dashboard**: List, activate, and deactivate projects with a single tap.
- **Terminal Emulator**: A live terminal with support for interactive stdin.
- **Cloud Integration**: Sync connection settings via Firebase.
- **Security**: Biometric authentication and secure token storage.

For setup details, see the [[Android Client Guide]].
