# IDE Plugin

The **IDE Plugin** for Android Studio and IntelliJ IDEA integrates the Omni Remote ecosystem directly into your development workflow.

## Features
- **Sidebar Integration**: Access your remote projects without leaving your IDE.
- **Remote Launch**: Trigger the workstation to open projects in a new Studio window.
- **Health Checks**: Monitor the status of your Remote Agent connection.

For setup details, see the [[IDE Plugin Guide]].
