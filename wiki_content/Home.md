# Welcome to the Omni Project Remote Wiki

Omni Project Remote is a professional-grade suite for managing and accessing your development workstation from anywhere.

## 🏗️ System Architecture
The system consists of three decoupled components communicating over a secure REST and WebSocket API:
1.  **[[Remote Agent]]**: The Windows backend and shell bridge.
2.  **[[Android Client]]**: The mobile management app.
3.  **[[IDE Plugin]]**: The integration for Android Studio/IntelliJ.
